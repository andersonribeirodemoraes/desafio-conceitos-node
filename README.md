
# Desafio Rocketseat

## Instalação e rodando o projeto
```sh
$ git clone https://github.com/andersonribeirodemoraes/desafio-conceitos-node.git
$ cd desafio-conceitos-node  
$ yarn
$ yarn dev

Rodando em: http://localhost:3333
```

## Rodando os testes
```sh
$ yarn test
```
